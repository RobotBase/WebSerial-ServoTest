📖 使用说明与技术文档 ▼

🚀 快速开始

  1. 选择波特率 (115200 用于 LX 直连协议)
  2. 点击 "🔌 选择串口并连接" → 浏览器弹出串口选择对话框
  3. 选择 USB 串口设备 (如 CH343)
  4. 连接成功后点击 "🔍 扫描舵机"
  5. 在舵机卡片中拖动滑块控制位置

⚠️ 安全注意事项

  • 电压: 工作电压 6~8.4V，建议 7.4V (2S 锂电)，切勿超压
  • 修改 ID: 总线上必须仅连接一个舵机
  • 接线: 先接信号线，再通电。断电后再拔线
  • 温度: 长时间高负载运行时注意散热，超过 60°C 应休息

📡 通信协议 (LX 直连)

指令 CMD 说明
移动 1 位置 0~1000, 时间 0~30000ms
写 ID 13 修改舵机 ID (0~253)
读 ID 14 广播地址 254 可读
读温度 26 返回 °C
读电压 27 返回 mV
读位置 28 返回 0~1000

帧格式: 0x55 0x55 ID LEN CMD [PARAMS] CHECKSUM

🧪 测试案例

  • 连接测试: 扫描后应显示位置/电压/温度
  • 移动测试: 滑块拖到 200→800→500，观察舵机转动
  • 卸载测试: 点击 🔓 后可手动旋转舵机轴
  • 图表测试: 实时曲线应持续更新约 2Hz
